# App Screen Specifications

## Screen 1: Live Prediction Card

**Trigger:** MatchMoment event, type = big_chance or corner_danger  
**Max display time:** Until moment resolves (8–15s) or fan taps  
**Surface:** Android / iOS bottom sheet, slides up over match stats

### Layout
```
┌─────────────────────────────────────┐
│  BIG CHANCE · xG 0.51              │
│                                     │
│  Will Saka score from here?         │
│                                     │
│  [    Goal    ]  [ No goal  ]       │
│                                     │
│  14,203 fans predicting · ~6s       │
└─────────────────────────────────────┘
```

### Resolution States
- Correct: green background flash, +35 pts badge, confetti (subtle)
- Incorrect: grey background, explanation stat ("Saka has missed 3 of his last 4 from this zone")
- Timeout (fan didn't tap): grey, "You missed this one" + correct answer

## Screen 2: Manager Mode

**Trigger:** Half time or 60th minute MatchMoment  
**Display duration:** Until fan picks or 3 minutes passes  
**Surface:** Full-screen takeover (match has paused)

### Layout
```
┌─────────────────────────────────────┐
│  YOU'RE THE MANAGER                 │
│  Arsenal trail 0–1. Odegaard is    │
│  off pace. Who do you bring on?     │
```

## Screen 3: Stat Whisper

**Trigger:** Sprint, goal, milestone MatchMoments  
**Display:** Non-interactive banner, auto-dismisses after 6 seconds  
**Surface:** Top notification bar style, slides down

### Layout
```
┌─────────────────────────────────────┐
│ ⚡ That was 36.8 km/h — faster than │
│   97% of PL sprints this season     │
└─────────────────────────────────────┘
```

Whispers should be emotionally resonant, not just statistical. Examples:
- "Saka has now been involved in 8 of Arsenal's last 11 goals"
- "That's the loudest the Emirates has been since the Derby in March"
- "Chelsea haven't scored from outside the box all season — until now"

## Screen 7: Wear OS Glanceable

**Single watch face complication during match**

```
┌──────────────┐
│  ARS  2–1    │
│  CHE  78'    │
│  xG ●●●○○   │
└──────────────┘
```

Single-tap on complication opens prediction card on watch. Haptic pulse on goal (2-pulse, 500ms gap).

```
│                                     │
│  Arsenal fans  ████████████░  84%   │
│  Chelsea fans  █████████░░░░  71%   │
```
