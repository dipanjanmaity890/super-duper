# 05 — Gamification System

## Point Economy

xG-adjusted scoring rewards knowledge, not luck. Predicting a 0.2 xG chance correctly requires more insight than a 0.8 xG chance — so harder predictions pay more.

| Action | Points |
|---|---|
| Prediction correct (high xG, >0.5) | +20 pts |
| Prediction correct (medium xG, 0.2–0.5) | +35 pts |
| Prediction correct (low xG, <0.2) | +50 pts |
| Manager sub matched real decision | +100 pts |
| Fan verdict correct (VAR, penalty) | +40 pts |
| Perfect match prediction (score + MOTM) | +500 pts |
| Streak bonus (3+ correct in a row) | ×1.5 multiplier |
| Match completion bonus (watched full 90) | +25 pts |
| First prediction of a match | +10 pts |

## Ethical Guardrails — This Is Not Gambling

This is the single biggest legal risk. The UK Gambling Commission has been aggressive about apps with prediction mechanics + reward systems.

**Non-negotiable rules:**
- Points have zero monetary value and cannot be exchanged for cash or betting credits
- No in-app purchases of any kind
- No "spend points to get another prediction"
- No countdown timers designed to create urgency
- No odds or probability display that resembles betting UI
- No partnerships with betting companies within the same app surface

**Pre-launch requirements:**
1. Legal review with specialist gambling counsel (Wiggin LLP or equivalent)
2. Copy-test every rewards screen — no language implying monetary value
3. Proactive Gambling Commission consultation (they offer informal guidance sessions)
4. App Store compliance review for both Apple and Google Play gambling policies

**The xG-adjusted scoring actually helps legally** — it rewards knowledge and pattern recognition, not luck. Document this explicitly in the legal submission.
